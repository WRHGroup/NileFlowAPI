Nile Basin Streamflow Explorer

  1. Download the HDF5 naturalized-flow file to your computer: naturalized flow file (.h5).
  2. Select that file with the “HDF5 flow file (.h5)” button below.
  3. Explore: click any element on the map — a sub-basin or a reach — or pick one from the dropdown, to view its streamflow.

The sub-basin and reach maps load automatically from this site. All computation runs locally in a background thread — your flow data never leaves your machine.
